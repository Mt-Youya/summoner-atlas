# Summoner Atlas 一期产品契约

## 范围

一期是一个基于公开、可核验数据的大乱斗版本决策工具，不宣称支持尚未接入数据源的区域、召唤师峡谷或竞技场。

- 数据范围：全球公开样本。
- 模式：大乱斗（ARAM）。
- 版本：由 `RESG_DATA_VERSION` 指定，默认 `16.13`。
- 实体：英雄、海克斯、英雄组合与海克斯协同。

## 目标路由与数据来源

| 路由                              | 用途                 | 数据来源                                 |
| --------------------------------- | -------------------- | ---------------------------------------- |
| `/zh`                             | 搜索、版本摘要、入口 | RESG 英雄/海克斯排行                     |
| `/zh/champions`                   | 英雄排行             | `champions/stats` + CommunityDragon 名称 |
| `/zh/champions/[id]`              | 英雄结论与组合       | RESG `base-stats`、`synergy`             |
| `/zh/augments`                    | 海克斯排行           | RESG `augments/tier-list`                |
| `/zh/augments/[id]`               | 海克斯详情           | RESG 海克斯排行                          |
| `/zh/combinations`                | 英雄组合研究入口     | RESG 英雄排行与详情接口                  |
| `/champion/[id]`、`/augment/[id]` | 旧链接               | 308 永久重定向                           |

## 非目标

- 账户、QQ/Riot 绑定、后台写操作与角色权限：没有 OAuth、会话与权限服务，不提供伪实现。
- 装备、符文、峡谷、竞技场、区域差异：等待真实字段与数据源后再开放。

## 验收标准

- 所有展示统计来自当前版本的 RESG 接口，失败状态明确可见。
- 英雄、海克斯搜索可到达对应详情；榜单 URL 中的筛选和排序可刷新恢复。
- 所有数据页显示“全球公开样本 / 大乱斗 / 版本”上下文。
- 旧英雄和海克斯链接返回永久重定向。
