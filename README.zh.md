# 召唤师档案

> 跨服务器的英雄联盟数据地图。

[English](README.md) · [한국어](README.ko.md)

召唤师档案（Summoner Atlas）是一个面向《英雄联盟》玩家的数据平台，帮助玩家从不同服务器、版本与游戏模式中快速理解英雄、装备、符文和玩法趋势。

项目当前处于重构规划阶段。完整的产品、设计与实施规范见 [REDESIGN_PLAN.md](REDESIGN_PLAN.md)。

![召唤师档案 Logo](public/brand/summoner-atlas-logo-concept.png)

## 产品目标

- 支持国服、韩服、美服、台服、欧服等多个服务器区域。
- 支持峡谷、大乱斗、竞技场及后续扩展模式。
- 提供英雄、海克斯、装备、符文、版本变化与组合研究数据。
- 让用户能更快获得带有样本量和版本上下文的可靠决策。

## 规划中的功能

- 全局搜索，支持英雄中文名、拼音、外号及海克斯名称。
- 英雄与海克斯排行榜，支持版本、区域、模式、定位、品质和排序筛选。
- 英雄详情，包含推荐方案、技能、装备、符文、海克斯与搭配数据。
- 海克斯详情与反查英雄。
- 组合研究和版本趋势分析。
- 用户账户、第三方账号绑定、偏好设置和管理后台。

## 技术栈

- Next.js 16
- TypeScript
- pnpm monorepo
- shadcn/ui，Base UI 基础组件
- Zustand
- Hugeicons

## 设计原则

- 结论优先，再提供完整原始数据。
- 跨设备可快速查询，移动端不压缩桌面表格。
- 明确展示版本、数据更新时间、样本量与可信等级。
- 使用可访问的语义、焦点和状态反馈。
- 保持原创视觉语言，不使用 Riot Games 官方标志或游戏素材作为品牌标识。
